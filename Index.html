App de controle financeiro pessoal Você quer algo simples, bonito e funcional. Abaixo está um app web com visual limpo, campos exigidos, totais (saldo, entradas, saídas), filtro mensal, acesso por link e opção de compartilhar. Salva os dados no navegador (localStorage), então você pode usar já, e publicar facilmente para acessar por URL. --- Recursos e fluxo • Campos: valor, categoria, data, descrição, tipo (entrada/saída). • Resumo: saldo total, total de entradas e total de saídas. • Visualização mensal: filtro por mês/ano com navegação rápida. • Compartilhar: botão que usa Web Share API e link da página. • Persistência local: dados ficam salvos no dispositivo via localStorage. • Acessibilidade: rótulos, contraste, foco visível, navegação por teclado. --- Código completo (HTML + CSS + JS) Crie um arquivo index.html com o conteúdo abaixo e publique.
